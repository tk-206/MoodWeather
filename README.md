# 🌈 MoodWeather

MoodWeather는 **기분과 날씨를 함께 기록**하는 감성적인 React 프로젝트입니다.  
단계별로 기능을 확장해가며, 프론트엔드 개발의 기본기부터 API 연동까지 학습하는 것을 목표로 합니다.

---

## 🚀 개발 단계별 목표

| 단계 | 이름 | 주요 내용 |
|------|------|-----------|
| Step 1 | 기분 기록 기본 기능 | 오늘의 기분 입력 및 저장 (localStorage) |
| Step 2 | 기록 리스트 보기 | 날짜별로 기분 기록을 리스트로 표시 |
| Step 3 | 데이터 유지 및 통계 기본 | 새로고침 시 데이터 유지, 주간 감정 통계 표시 |
| Step 4 | 날씨 API 연동 | OpenWeather API로 현재 날씨 표시 |
| Step 5 | 감정 + 날씨 연동 | 기분과 날씨를 함께 저장 및 통계화 |
| Step 6 | UI 개선 | 감성적인 테마, 이모지 중심 UI, 반응형 디자인 |

---

## 🛠️ 기술 스택

- **React + Vite**
- **TailwindCSS**
- **LocalStorage**
- (후에 추가) **OpenWeather API**
- (후에 추가) **Chart.js / Recharts**

---

## 🎯 학습 포인트

- 컴포넌트 구조 설계 및 상태 관리
- 로컬 데이터 저장(LocalStorage)
- API 연동 (fetch / async-await)
- UI 개선 및 CSS 모듈화
- React hooks (useState, useEffect) 활용
